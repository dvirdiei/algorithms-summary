  <article className="conceptCard conceptCard--algorithm">
    <p className="conceptCard__label">אילוצים</p>
    <h3 className="conceptCard__title">מהי זרימה חוקית?</h3>
    <p className="conceptCard__body">
      <strong>1. מגבלת קיבולת:</strong>
      <br/>
      0 ≤ f(u,v) ≤ c(u,v)
      <br/><br/>
      <strong>2. שימור זרימה:</strong>
      <br/>
      לכל v ≠ s,t:
      <br/>
      סכום נכנס = סכום יוצא
      <br/><br/>
      (מה שנכנס - יוצא, לא "נעלם")
    </p>
  </article>
  <article className="conceptCard conceptCard--solution">
    <p className="conceptCard__label">מטרה</p>
    <h3 className="conceptCard__title">זרימה מקסימלית</h3>
    <p className="conceptCard__body">
      <strong>|f| = סכום הזרימה היוצאת מ-s</strong>
      <br/><br/>
      המטרה: למצוא זרימה חוקית f שממקסמת את |f|
      <br/><br/>
      <strong>משפט:</strong> |f| = סכום הזרימה הנכנסת ל-t
      <br/>
      (כי הכל משתמר בדרך)
    </p>
  </article>
</section>
