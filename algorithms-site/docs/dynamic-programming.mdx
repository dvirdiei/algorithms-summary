         <p>
            \(dp[i][w]\) — ערך מקסימלי עם \(i\) פריטים ראשונים וקיבולת \(w\).
         </p>
         <p>
            $$
            dp[i][w] = \max\left(dp[i-1][w],\ value_i + dp[i-1][w-weight_i]\right)

            כאשר \(weight_i \leq w\).
         </p>
         <ul>
            <li>טבלה בגודל \((n+1) \times (W+1)\).</li>
            <li>לשיפור זיכרון: עדכון חד-ממדי מהקיבולת הגבוהה לנמוכה.</li>
